# AI Chat Enhancer

Chrome extension that supercharges ChatGPT, Claude, and DeepSeek with templates, folders, export, and more.

## Features

- **Templates** — 25 built-in + custom, with `{{variable}}` support and drag-to-reorder
- **Folders** — organize chats into colored folder trees
- **Export** — save conversations as Markdown, Text, JSON, or PDF
- **Search** — full-text search across all conversations
- **Auto-save** — drafts saved every 1.5 seconds
- **Keyboard shortcuts** — quick access to all features

## Supported Platforms

- ChatGPT (chatgpt.com)
- Claude (claude.ai)
- DeepSeek (chat.deepseek.com)

## Development

```bash
# Load unpacked extension in Chrome
1. Open chrome://extensions
2. Enable "Developer mode"
3. Click "Load unpacked" → select this directory
```

No build step. Pure vanilla JS, Manifest V3.
